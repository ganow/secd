# SECDマシン

### 実行方法

予め用意したスクリプトの実行

	python secd.py /path/to/script

対話モードの起動

	python secd.py

### 受けつける構文

ex)

	((lambda x
	    (lambda y
	        (x y)))
	    ((lambda x
	        (lambda y
	            (x y)))
	        (lambda x
	            (lambda y
	                y))))

ラムダ式を記述する際は((lambda 束縛変数 本体) 実引数)のように記述し、
スペースかもしくは改行とタブで間を区切る。
なお、括弧を省略する記法は用いることができない。

### 対話モード

対話的に入力文字列を受け取って結果を返すことができる。
このモードでも複数行にわたっての入力が可能であり、
改行のみを入力した時点でそれまでのコードが実行される。
